Fasilkom-TI Laksanakan Kuliah Praktisi Bidang Virtual Systems: Practical Guide to Cloud Virtualization, IaC, and Security
Dipublikasi Pada

19 Juni 2024

Dipublikasi Oleh

Ahmad Fadhil S.Kom

Thumbnail Fasilkom-TI Laksanakan Kuliah Praktisi Bidang Virtual Systems: Practical Guide to Cloud Virtualization, IaC, and Security
Fasilkom-TI USU mengadakan kuliah praktisi tentang cloud virtualization dan keamanan pada 6 Juni 2024, dengan narasumber Ryan Hamonangan, diikuti sesi tanya jawab yang antusias.

Fakultas Ilmu Komputer dan Teknologi Informasi (Fasilkom-TI) Universitas Sumatera Utara (USU) telah melaksanakan kuliah praktisi bidang virtual systems dengan judul “Practical Guide to Cloud Virtualization, IaC, and Security”. Kuliah praktisi ini dilaksanakan pada Kamis, 6 Juni 2024 di Digital Learning Centre Building (DLCB) Lantai 8 Universitas Sumatera Utara.

Kegiatan ini dimoderatori oleh Dosen Teknologi Informasi, Seniman, S.Kom, M.Kom., dilanjutkan kata sambutan Ketua Prodi Teknologi Informasi, Dedy Arisandi, S.T, M.Kom., dan menyanyikan lagu Indonesia Raya. Narasumber kuliah praktisi ini adalah Ryan Hamonangan, S.Kom, M.T., seorang Software Architect dari Astra International Tbk Olx Indonesia.

Pada pemaparan, Ryan menjelaskan tentang cloud computing, seperti penyedia layanan cloud computing yang banyak dipakai oleh orang-orang, bagaimana cara kerjanya dan manfaat penggunaan dari cloud computing. Selain itu, Ryan juga mengajak para mahasiswa untuk membuat rancangan database untuk website USU cafe. Selanjutnya, pemberian sertifikat dari Fasilkom-TI kepada narasumber.

Di akhir sesi, dibuka tanya jawab kepada narasumber. Beberapa mahasiswa Fasilkom-TI bertanya kepada narasumber terkait penggunaan database, bagaimana suatu sistem terkena kebocoran data dan blockchain. Dari sesi tanya jawab ini, menunjukan bahwa para mahasiswa Fasilkom-TI sangat antusias dengan kuliah praktisi ini.
