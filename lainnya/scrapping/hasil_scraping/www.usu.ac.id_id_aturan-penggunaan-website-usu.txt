Aturan Penggunaan

Departemen, fakultas, dan unit kerja bertanggung jawab untuk membuat dan memperbarui halamannya. Selalu perbarui informasi di situs web terkait.
Departemen, fakultas, dan unit kerja harus menyediakan website versi bahasa Inggris.
Website jurusan, fakultas dan unit kerja harus memiliki RSS feed. Jika situs web menggunakan fungsionalitas WordPress USU, umpan RSS akan dibuat secara otomatis. Jika website jurusan/fakultas/unit kerja dibuat sendiri, pastikan memiliki kemampuan RSS feed.
Waspadai semua masalah hak cipta. Pastikan Anda memiliki izin untuk menggunakan materi yang Anda posting dari penulis atau fotografer yang relevan. Pastikan bahan diberi label dengan benar. Tinjau juga pemberitahuan hukum di halaman hak cipta/properti dan berikan tautan ke halaman ini jika Anda memerlukannya untuk situs Anda.
Setiap jurusan, fakultas atau unit kerja harus memiliki minimal satu orang yang bertanggung jawab secara keseluruhan atas isi website terkait (administrator website).
Administrator situs web bertanggung jawab untuk memastikan bahwa penulis materi universitas selalu memperbarui materi dan tidak memposting materi yang tidak pantas di situs web lembaga pendidikan. Penulis harus selalu berasumsi bahwa semua konten di situs web yang dihosting di server universitas dapat bersifat publik (akses gratis untuk semua), meskipun sebenarnya tidak ditujukan untuk publik. Penting untuk menjaga agar materi tetap mutakhir dan tersedia. Perubahan pada halaman ini tidak dapat dilakukan tanpa pemberitahuan dari entitas yang bertanggung jawab atas manajemen konten. Untuk informasi lebih lanjut, hubungi Help Desk/Call Center USU.
Operator situs web bertanggung jawab untuk memastikan bahwa situs web mereka dapat diakses oleh pengguna penyandang disabilitas yang dapat menggunakan teknologi pendukung (misalnya pembaca layar atau pembesar layar) untuk membaca situs web. Informasi tentang studi dan program universitas harus dapat diakses oleh semua orang. Selain itu, unsur program perkuliahan dan materi yang ditawarkan secara online harus mudah diakses.
Jika informasi yang ditampilkan berkaitan dengan waktu, tanggal efektif harus dinyatakan dengan jelas (misalnya "biaya kuliah 2022-2023").

Universitas Sumatera Utara memiliki dedikasi untuk mengelola jaringan yang dapat diakses, yang akan memberikan civitas universitas akses penuh ke informasi universitas, program kuliah, dan kegiatan yang ditawarkan melalui jaringan. Universitas juga berkomitmen untuk menerapkan standar aksesibilitas situs web minimum dan mendorong penerbit online untuk mematuhi standar aksesibilitas. Jika Anda menggunakan teknologi dan materi pendukung di situs web kami yang mencegah akses ke informasi, silakan hubungi Help Desk/Call Center USU.
