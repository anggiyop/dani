Balakka Peel-Off Mask
Peel-Off Mask
Product Description

Balakka Peel-off Mask is a peel-off mask that is efficacious as an anti-aging. This mask is made from balakka fruit ethanol extract, which does not use SLS because balakka fruit ethanol extract contains saponins that function as cleansers and foam producers. The mask does not use synthetic antioxidants because balakka fruit ethanol extract contains flavonoids which are natural antioxidants, and does not contain synthetic antibacterials. After all, balakka fruit ethanol extract contains alkaloids and anthraquinones as natural antibacterials.
